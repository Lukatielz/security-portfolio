\# SIEM Detection Lab



\## Objective

Set up a SIEM (Splunk/ELK) to ingest logs from a lab VM and detect suspicious activity, such as repeated failed login attempts.



\## Tools Used

\- Splunk Free / ELK Stack

\- Windows 10/11 VM (log source)

\- VirtualBox



\## Steps

1\. Configured Windows VM to forward Event Logs to Splunk

2\. Wrote a search/detection rule for brute-force login attempts

3\. Verified alerts triggered correctly
